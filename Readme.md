## About The Project

This is a Restful API repository for Shop Id. This Restful API is built using ExpressJS and PostgreSQL.

## Installation

- npm install or npm i
- npm run server

## Debug

- npm run lint

### Executing program

- Run program with `npm run dev` for development and `npm run start` for production
